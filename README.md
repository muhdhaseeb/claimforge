# ⚒ ClaimForge™
### AI-Powered Patent Claim Drafter

> *"From idea to claim, in minutes."*

ClaimForge is a free, browser-based tool that helps solo inventors and small engineering teams draft USPTO-style patent claims using AI — without needing a patent attorney for the initial draft.

---

## 🎯 What It Does

Inventors describe their invention in plain English. ClaimForge generates a full set of professionally structured patent claims using proper USPTO legal language — ready to hand to an attorney or attach to a provisional patent application.

**A patent attorney charges $8,000–$15,000 per filing. ClaimForge generates a first draft in under 60 seconds, for free.**

---

## ✨ Features

- **Real AI generation** — powered by Groq API (Llama 3.3 70B), not templates
- **USPTO-compliant language** — `comprising`, `wherein`, `operably coupled to`, `configured to`, `plurality`, `further comprising`
- **Smart claim hierarchy** — Independent apparatus claim + dependent claims + independent method claim
- **8 technology categories** — Mechanical, Electrical, Software, Biomedical, Chemical, Consumer, Agricultural, Energy
- **Category-aware language** — electrical inventions never get polymer/alloy materials claims
- **Logically correct role assignment** — sensors monitor, actuators perform, controllers regulate
- **Legal term highlighting** — key patent terms highlighted in gold throughout the vault
- **Claim Scope meter** — visual indicator of how broad your protection is
- **Copy to clipboard** — one-click export of all claims as plain text
- **Grammar auto-fix** — client-side correction of common AI patent grammar errors
- **Fallback engine** — if AI fails, built-in template generator ensures you always get claims
- **XSS protection** — all user inputs sanitized before rendering
- **Zero backend** — single HTML file, runs entirely in the browser

## 🚀 Quick Start

### 1. Get a Free Groq API Key
1. Go to [console.groq.com/keys](https://console.groq.com/keys)
2. Sign in with Google — one click, no credit card required
3. Click **"Create API Key"** → name it anything → **Submit**
4. Copy the key (starts with `gsk_`)

### 2. Open ClaimForge
- Download `claimforge.html`
- Serve it via a local server (required for API calls):

**Python:**
```bash
python -m http.server 8080
```
Then open [http://localhost:8080/claimforge.html](http://localhost:8080/claimforge.html)

**VS Code:**
Install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) → right-click `claimforge.html` → **Open with Live Server**

**Node.js:**
```bash
npx serve .
```

### 3. Paste Your Key & Invent
Paste your Groq API key into the verification screen and start drafting.

## 🖥️ How It Works

```
User describes invention (title, problem, components, novelty)
        ↓
ClaimForge builds a USPTO patent attorney system prompt
        ↓
Groq API (Llama 3.3 70B) generates claim JSON
        ↓
Client-side grammar auto-fix runs on all claims
        ↓
Claims rendered in the Vault with legal term highlighting
        ↓
User copies claims for filing or attorney review
```
